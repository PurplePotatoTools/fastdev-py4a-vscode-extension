# fastdev-py4a-vscode-extension
借助一些非常规方式实现在vscode中使用python开发android app, 适用个人工具或者原型的快速实现, 谨慎用于生产环境, 无法在google play等应用商店发布


## 特点
- 无需安装Android Studio, 直接使用vscode开发
- 无需root权限, 使用proot进入debian环境来启动python服务端程序
- python版本3.11
- 使用webview实现界面
